This directory contains icons for Progressive Web App (PWA) support and favicons for various devices.

## Required Icons

Create the following icon files from your logo or a professional portrait:

### PWA Icons (for manifest.json)
- `icon-72x72.png` - Small Android icon
- `icon-96x96.png` - Standard Android icon
- `icon-128x128.png` - Android icon
- `icon-144x144.png` - Android icon
- `icon-152x152.png` - iOS icon
- `icon-192x192.png` - Standard PWA icon
- `icon-384x384.png` - Large PWA icon
- `icon-512x512.png` - High-res PWA icon

### Browser Favicons
- `icon-16x16.png` - Browser tab icon (small)
- `icon-32x32.png` - Browser tab icon (standard)
- `apple-touch-icon.png` (180x180) - iOS home screen icon

## Design Guidelines

- Use a simple, recognizable design (logo, initials "KC", or professional headshot)
- Ensure the design works at small sizes
- Use transparent background for PNG files
- Consider using a solid background color (#2c2c2c or #d4af37) for better visibility
- Keep design centered with appropriate padding

## How to Create Icons

### Option 1: Use an Online Generator
1. Use a tool like [Favicon Generator](https://realfavicongenerator.net/)
2. Upload a square image (at least 512x512px)
3. Download the generated icon pack
4. Place icons in this directory

### Option 2: Manual Creation
1. Create a 512x512px design in Photoshop/Figma
2. Export at various sizes listed above
3. Ensure proper naming matches the manifest.json references

## Placeholder Icons

If you don't have icons ready yet, the website will function without them. However, for a professional PWA experience, custom icons are highly recommended.
